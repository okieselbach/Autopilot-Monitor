# Autopilot Monitor Web

Web dashboard for Autopilot Monitor, built with Next.js, React, and Tailwind CSS.

## Setup

```bash
npm install
cp .env.local.template .env.local
npm run dev
```

Configure `NEXT_PUBLIC_API_BASE_URL` in `.env.local` to point to the backend API.

## Scripts

- `npm run dev` - Development server
- `npm run build` - Production build
- `npm run lint` - Linting
